# 特性

- 生成符合 Golang 设计哲学的工程框架，包括基础功能(JWT, OpenTracing, ZapLog, Promtheus)
- 根据数据库生成对应 CRUD 方法
- 支持查询过滤、排序、分页、指定返回字段、批量查询、批量删除
- 支持 BelongTo \ HasOne \ HasMany \ ManyToMany 关联配置
- 支持配置校验信息
- 支持 Swagger 文档自动生成
- 支持业务扩展
- 表结构更改可多次生成代码
- 资源内嵌，一个二进制文件即可启动
